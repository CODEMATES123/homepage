活動概要

CODE MATESはIT技術を活用し社会に貢献することを目的としています。「たまっぷ」に代表されるプロジェクトを通じてIT技術の向上、そして社会貢献に努めています。

プロジェクト紹介

たまっぷ（外部サイトにつながります）

「たまっぷ」は「CODE MATES」が最初に開発したWebアプリケーションです。法政大学と最寄り3駅間のバス発着情報を提供するものです。 TypeScriptとReactを使っています。静的ホスティングでありながらフロントエンドに機能を集約することで高いUXを実現しています。

たまらぼ

「たまらぼ」について（法政大学のホームページ）

「たまらぼ：通称」は社会問題を解決したい同世代の学生が集うプログラム「チェンジメーカーズラボ in 多摩」です。
「CODE MATES」として、IT技術を駆使して本学周辺地域の利便性向上に貢献したいと考えています。

たまらぼ DEMO DAYの様子

HoseiLink

「HoseiLink」はCODE MATESが開発した法政大学生向けSNSアプリです。同じ学部・学科や同じ授業を登録している人同士でつながることができます。法政大学の学生のみ閲覧・利用ができるようになっています。

その他

月・火・金曜日の15：30～活動しています！

使っている・使う予定の技術

Python
近年AIの需要が高まるにつれて授業が急増している言語です。AI・データサイエンス分野に向いています。
HTML
Webページの「骨組み」をつくための言語です。厳密にはプログラミング言語ではありません
CSS
Webページのデザインを作るための言語です。HTMLと共に使用してWebページを形作っていきます。
JavaScript
Webブラウザで動作するプログラミング言語です。
TypeScript
JavaScriptに「型」の概念を導入した言語です。
React
動きのあるWebサイトを作る際に用いられるJavaScriptのフレームワークです。
Next.js
Reactと組み合わせてWebサイトのフロントエンドとバックエンドを作ることができます。
C#
「Unity」というソフトウェアとともに使ってゲームを作ることができます。
Java
金融，行政のような日本の多くの基幹システムで使われている言語です。
