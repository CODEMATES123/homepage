活動予定

　CODE MATESはIT技術を活用し社会に貢献することを目的としています。「たまっぷ」に代表されるプロジェクトを通じてIT技術の向上、そして社会貢献に努めています。

プロジェクト紹介

たまっぷ

たまっぷ（外部サイトにつながります）

　「たまっぷ」はCODE MATESが最初に開発したWebアプリケーションです。法政大学とその最寄り駅間のバスの時刻をリアルタイムで表示しています。
　言語はPythonを使用していてサーバ側ではFlaskを使っています。ユーザ側ではサーバから送られてきたデータを保持しています。JavaScriptを使ってユーザの操作に応じて適切に表示を切り替えています。

たまらぼ

「たまらぼ」は社会問題を解決する人たちが集うプログラム「チェンジメーカーズラボ in 多摩（通称：たまらぼ）」です。
　CODE MATESとしてはIT技術を活用して法政大学周辺の地域課題を解決していきたいと考えています。
